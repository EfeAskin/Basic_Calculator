# 🧮 C Calculator

A simple calculator application written in C.  
This project performs basic arithmetic operations from user input via the terminal.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- User input handling
- Basic error control (e.g., division by zero)

## 🛠️ Technologies Used

- C Programming Language
- Standard C Library (stdio.h, stdlib.h)

## 📂 How to Run

### 1️⃣ Clone the repository

    ```bash
    git clone https://github.com/EfeAskin/tic-tac-toe-c.git
    cd tic-tac-toe-c
### 2️⃣ Compile the code
    gcc calculator.c -o calculator
### 3️⃣ Run the program
    ./calculator
